# 9. Glossário rápido

Termos que aparecem no dia a dia do SagaOne.

| Termo | Significado |
|---|---|
| **Agente** | Número WhatsApp cadastrado no sistema (Paty ou Pri). Um agente atende uma marca + UF. |
| **Cadência (WPP)** | Sequência de 4 templates (convite, lembrete, D-1, D-0) obrigatória quando a flag está ativa. |
| **Cadência (Paty)** | Sequência conversacional de follow-ups pós-venda. |
| **Check-in** | Status que marca a chegada do cliente na loja. Dispara integração com MobiGestor. |
| **CRM (papel)** | Time responsável por bases, quarentena e configuração da Paty. |
| **Empresa ativa** | Empresa selecionada no topo. Define o contexto de tudo. |
| **Evento base** | "Container" da campanha — nome, período, loja. |
| **Feature flag** | Chave global que liga/desliga funcionalidade. Mexida por Admin/TI. |
| **Gatilho** | Evento externo (ex.: "entrega confirmada") que pode disparar template Paty. |
| **Job órfão** | Disparo que travou em processamento. Sistema recupera sozinho em ≤15 min. |
| **Kanban** | Tela principal da Prospecção. Colunas = status. |
| **Master** | Perfil com acesso a todas as empresas e telas. |
| **MobiGestor** | Sistema externo de atendimento (recebe o lead após check-in). |
| **Paty** | Agente WhatsApp de pós-venda. Roda fora do SagaOne. |
| **Pool / DataLake** | Base interna do Grupo Saga (últimos 12 meses de clientes). |
| **Pri** | Agente WhatsApp de prospecção. |
| **Prospecção** | Módulo de convite/qualificação de lead (pré-venda). |
| **Quarentena** | Bloqueio de telefone por marca + canal. Evita spam ao mesmo cliente. |
| **SDR** | Perfil que trabalha topo do funil. Limite de 30 leads em aberto. |
| **Template** | Mensagem de WhatsApp aprovada pela Meta. |
| **Template pausado** | Meta bloqueou o template por reclamações. Precisa trocar. |
| **Temperatura** | Prioridade do lead: quente / morno / frio. |
| **Vault** | Cofre de segredos criptografados. Acesso via MFA. |
