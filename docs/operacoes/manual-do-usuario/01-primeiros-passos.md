# 1. Primeiros passos

## Login

- O SagaOne aceita apenas contas Microsoft do domínio **`@gruposaga.com.br`**. Não há cadastro por e-mail/senha.
- Ao entrar, clique em **"Entrar com Microsoft"** e use a conta corporativa. Se aparecer tela de senha comum, você está no login errado.
- Se você acabou de ser cadastrado e o botão dá erro, avise TI — o usuário precisa estar liberado no Azure antes do primeiro login.

## Empresa ativa

- No topo direito existe o **seletor de empresa**. Ele define o contexto de **tudo** que você vê (leads, eventos, relatórios, templates).
- Sempre confira a empresa ativa antes de disparar campanha, subir planilha ou criar evento. É a fonte número um de erro operacional.
- Quem trabalha em várias marcas alterna aqui. Master vê todas.

## Sessão e inatividade

- A sessão dura até **8 horas**. Depois disso, é preciso logar de novo.
- Se ficar **1 hora sem interagir**, o sistema desloga automaticamente por segurança.
- Se atualizar a página e sumir dados, geralmente é sessão renovada — basta esperar 2 segundos e tentar de novo.

## Menu lateral

- Só aparecem os módulos que seu perfil pode acessar. Se está faltando algo, não é bug — é permissão (fale com Admin/TI).
- Módulos principais: **Prospecção**, **Recepção**, **Pós-Vendas**, **Resultados**, **Administração**.

## Se algo der errado

| Sintoma | O que fazer |
|---|---|
| Login com Microsoft dá erro | Confirmar com TI se seu usuário está liberado no Azure. |
| Não vejo minha loja | Trocar a empresa ativa no topo. |
| Fui deslogado de repente | Provavelmente passou 1h ocioso — logar de novo, é normal. |
| Menu está diferente do colega | Perfil de acesso diferente — não é bug. |

> 🎥 Vídeo sugerido: *"Primeiro acesso ao SagaOne"* (P0 — pendente de gravação).
